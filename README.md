Monorepo for JavaScript and TypeScript DX libraries.

## Packages

- [`eslint-plugin-function`](./packages/eslint-plugin-function) - (WIP) an ESLint plugin for function-related rules
- [`eslint-plugin-function-rule`](./packages/eslint-plugin-function-rule) - an ESLint plugin for writing custom rules with JavaScript functions
- [`tsl-dx`](./packages/tsl-dx) - A tsl plugin for better JavaScript/TypeScript DX
- [`tsl-rules-of-react`](./packages/tsl-rules-of-react) - Rules of React for tsl
- [`tsl-shared`](./packages/tsl-shared) - Shared utilities for tsl
